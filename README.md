# Test technique Angular/PHP

Application frontend Angular permettant d’interagir avec une API PHP fournie.

## Installation

### Backend

Le backend PHP (`index.php`, `.htaccess`) n’est **pas inclus** dans ce repository.

Pour faire fonctionner l’application, il est nécessaire de :

1. Mettre en place l’API fournie dans un environnement PHP local (WAMP, XAMPP, Laragon etc.)
2. Rendre l’API accessible à l’adresse : http://localhost

### Frontend

Installer avec npm :

```bash
npm install
```

Lancer l'application :

```bash
ng serve
```
